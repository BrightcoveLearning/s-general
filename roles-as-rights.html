Introduction

Using the Roles as Rights feature, you can present specific content to viewers based on their assigned roles. The following content types can be shown or hidden based on the viewer's role:

  • Movies
  • Programs
  • Seasons
  • Series
  • Episodes
  • Channels
  • Live Events
  • Playlists
  • Pages

In the Beacon CMS, you are able to create roles and assign them to each each content type listed above.

You will then use your OIDC roles claim URL to associate viewers with roles. OpenID Connect (OIDC) is an authentication protocol used to verify the identity of a user to a client service.

For details about using OIDC, see the OIDC Authentication Configuration and Testing document.
