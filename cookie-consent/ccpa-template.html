Create the CCPA Template

  1. In the left navigation, click Templates.
  2. Click Add New
  3. Select the CCPA Template (California) for the Framework.
  4. Click Next.
  5. Complete the form then click Create.
  6. Customize Web Template, Customize CTV Template, Customize Mobile Template.
  7. Click Save Template.

Customizing The Web Template

You can customize the layout, styling, content, and behavior of your banner template. This allows you to create a look and feel that matches your organization's branding style and behavior preferences.

  1. After you created your CCPA template, the Template Details page opens to the Banner tab. Click Preference Center.
  2. In the Preference Center:
    1. Open the Styling options
    2. Enable Override Banner Styling
    3. Upload a new logo
      Note: The optimum size for a logo in the Preference Center is: Height: 50px ; Width: 140px
    There is no need to make any other styling changes here as the banner and preference center will automatically be styled based on the design of your Beacon app.
  3. Click Save Template.

Customizing The CTV Template

You can customize the layout, styling, content, and behavior of your banner template. This allows you to create a look and feel that matches your organization's branding style and behavior preferences.

  1. On the COOKIE CONSENT menu, select Templates.
  2. Click on the name of the template you want to customize.
    1. Select the Details tab.
    2. Enable CTV App Template.
    1. Select the Banner tab.
    2. On Platform select CTV App.
    3. On design, Edit the Button Sets with the colors of your Brightcove Beacon Design Guide.
  5. Save and Close
    1. Select the Preference Center tab
    2. On Platform select CTV App.
    3. On Design, Edit General with the colors of your Brightcove Beacon Design Guide.
    4. On Design, Edit Button Sets with the colors of your Brightcove Beacon Design Guide.
  7. Save and Close
  8. Save Template
Note: Every time you make changes to the styling of a template within One Trust, you have to publish the SDK to ensure that all changes are properly implemented and accessible for seamless integration.

Customizing The Mobile Template

You can customize the layout, styling, content, and behavior of your banner template. This allows you to create a look and feel that matches your organization's branding style and behavior preferences.

  1. On the COOKIE CONSENT menu, select Templates.
  2. Click on the name of the template you want to customize.
    1. Select the Details tab.
    2. Enable Mobile App Template.
    1. Select the Preference Center tab
    2. On Platform select Mobile App.
    3. On Design, Edit General with the colors of your Brightcove Beacon Design Guide.
  5. Save and Close
  6. Save Template
Note: Every time you make changes to the styling of a template within One Trust, you have to publish the SDK to ensure that all changes are properly implemented and accessible for seamless integration.
